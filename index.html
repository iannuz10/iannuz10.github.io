Developed a C++ framework to solve integrated Task and Motion Planning (TMP) problems for multi-robot systems. The solution combines a PDDL-based task planner with a custom motion planner using Dijkstra's algorithm and a novel collision avoidance strategy.

Course Project: Control of an Inverted Pendulum

Designed and implemented a full state-space controller for the classic inverted pendulum problem. Using MATLAB & Simulink, I developed a state feedback controller via pole placement and a Luenberger observer to stabilize both linear and non-linear models of the system.

Key Tech: MATLAB, Simulink, Control Theory, State-Space
